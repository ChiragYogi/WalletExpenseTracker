## Features


- Simple UI, Easy To Use, Clean and Intuitive Expense & Income Tracker
- Light Theme /Dark Theme
- Intuitive Pie Chart For Monthly Spending
- Weekly, Monthly, Yearly Detail View
- Calendar View
- Reminder Notification
